DEVELOPER BUILD #26

Эта записка создана для игроков, которые не понимают, что делать в этой игре.

-- ЦЕЛЬ ИГРЫ --

Вкратце, цель игры - пройти настолько много циклов насколько возможно. Чтобы пройти цикл, вам нужно уничтожить
торговый корабль Небесный (на космическом поле отмечен оранжевой меткой) и добраться до портала к следующему циклу.
Между циклами игрок может улучшить свой корабль.

-- ВРАГИ --

(Пока...) Враги отличаются методом обнаружения игрока. Клетки, на которых враг обнаруживает игрока, подсвечены.
Если игрок оказывается в поле зрения нескольких врагов, ему придется сразиться с несколькими одновременно.

-- КРЕДИТЫ --

Кредиты выпадают из врага, если в него попадают пули и после его уничтожения. Они нужны для покупки улучшений.

-- ЧИТЫ --

В игре есть читы. Используйте их в научно-учебных целях, пожалуйста!
Для активации режима читов нажмите ENTER (Return) с зажатой клавишей ALT (любой из двух).
Для выключения сделайте то же самое.
Список читов (активируются при нажатии указанных клавиш):
    ALT+N | Невидимость для военных кораблей
    ALT+M | +100 кредитов
    ALT+D | Урон себе
    ALT+A | Урон врагу
    ALT+J | Убить себя
    ALT+K | Убить врага
