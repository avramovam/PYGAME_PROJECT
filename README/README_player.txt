-- ЧИТЫ --

В игре есть читы. Используйте их в научно-учебных целях, пожалуйста!
Для активации режима читов нажмите ENTER (Return) с зажатой клавишей ALT (любой из двух).
Для выключения сделайте то же самое.
Список читов (активируются при нажатии указанных клавиш):
    ALT+N | Невидимость для военных кораблей
    ...
